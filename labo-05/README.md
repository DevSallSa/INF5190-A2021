## Environnement virtuel

1. Créer l'environnement virtuel

```bash
# project root
python -m labo05-env && source labo05-env/bin/activate
```

2. Installer les requirements

```bash
pip install -r requirements.txt
```

3. Démarrer Flask

```bash
make
```
